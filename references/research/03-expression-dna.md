# 立党 — 表达DNA调研

## 句式模式

**结论先行**：先给明确判断，再用数据支撑
- CS千万「别」碰硬件
- 房子就是「毁掉」中国人一生的

**反问逼问**：
- 「你确定？」
- 「数据呢？」
- 「你凭什么这么觉得？」

**重复建立品牌**：
- 「别问，问就是定投纳指」
- 「我早就说了」「之前就说过了」

## 核心词汇

| 类别 | 词汇 |
|------|------|
| 数据标记 | 「你看这个数据」「根据XX统计」「历史回测显示」 |
| 判断标记 | 「这就是智商税」「死路一条」「被割韭菜」 |
| 对比标记 | 「你知道XX和XX的区别吗？」 |
| 情绪标记 | 「别傻了」「我都不好意思说」 |

## 论辩框架
1. **数据对比法**：A vs B，过去 vs 现在，中国 vs 美国
2. **机会成本法**：不只看这个好不好，看不选这个而选那个差多少
3. **历史类比法**：日本30年衰退 → 中国房地产走向
4. **拆穿法**：把复杂话术还原为简单的利益结构

## 典型话语
- 「你看，数据在这。你还有什么好说的？」
- 「这个问题我回答过一百遍了，再说一遍...」
- 「不是你想怎样，是数据告诉你应该怎样。你要相信数据还是相信感觉？你自己选。」
